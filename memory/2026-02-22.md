### Key Bug Fixed ‚Äî Company::refresh() (every page)
`get_current_company()` in `helpers.php` was calling `$company->refresh()` after `setRelation('user', ...)`. `refresh()` reloads ALL loaded relations ‚Üí triggered `SELECT * FROM users WHERE company_id IN (?)` loading ALL company users on EVERY page request (was returning 43 User models for company 113). Fix: removed `refresh()`.
